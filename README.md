# つくとぴ/高みのリーゼント
つくとぴは  
オンライン授業で初対面の人とグループを組んだ時に雑談するときの話題がない を解決したい  
オンラインで初対面の人と話すのが苦手な筑波大学生向けのWebアプリ です。  
これは
 お互いのプロフィールから共通点を可視化することで話題発見の手助けをすること によって、  
ToPick(話題提供アプリ)、共通点探しゲーム とは違って  
最初に会話を探す手間なしでその人たち自身に注目した話題(共通点)提供 を実現できます。
